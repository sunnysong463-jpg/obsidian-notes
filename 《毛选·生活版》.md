# 🎬 系列名：《毛选·生活版》

**核心设定：**  
Sunny = 普通年轻人，用生活问题“撞上理论”

**每集结构统一（强烈建议固定）：**

1. 问题出现
2. 错误应对
3. 崩溃/冲突
4. 理论转化（关键）
5. 行动改变（落地）

---

# 01｜实践论：光想不做=没用

《毛选》的内容本身偏理论+历史语境，如果只是读，很容易变成口号化或空泛表达，反而不好理解。结合生活，把抽象思想拆成生活场景，更容易实践

**主题：拖延症**

- 第一格：  
    Sunny列计划（健身/早起/学习）  
    旁白：明天开始新人生！
- 第二格：  
    躺平刷手机  
    Sunny：再看一会…
- 第三格：  
    一天过去，啥都没做  
    音效：滴答滴答
- 第四格：  
    开始只做一件小事（穿运动鞋）  
    旁白：先动起来，再优化

---

# 02｜矛盾论：抓主要矛盾

**主题：事情太多焦虑**

- 第一格：  
    10个待办事项压头
- 第二格：

- 第三格：  
    截止时间爆炸
- 第四格：  
    只选最重要1件先做  
    旁白：先解决最关键的
